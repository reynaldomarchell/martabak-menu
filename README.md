## Download

1. Clone this project

```bash
  git clone https://github.com/reynaldomarchell/martabak-menu.git martabak-menu
```

2.  Go to the project directory

```bash
  cd martabak-menu
```

3. Install the dependencies

```bash
  npm install
```

4. Start the app

```bash
  npm start
```
